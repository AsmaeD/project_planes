Les données

Notre base de données se compose de 4 tables.

Deux tables rassemblent des informations sur les différents aéroports dans le monde et deux autres rassemblent différentes informations sur les accidents aériens qui ont eu lieu de 1919 à nos jours.

Nous vous invitons à naviguer à travers ces tables pour visualiser les données.

Informations sur les tables (1)

Les tables Airports_dep et Airports_arr sont construites de la même manière:

ICAO : code caractérisant un unique aéroport dans le monde (clé primaire)

Name_airport : Nom officiel de l'aéroport

City : Ville dans laquelle se situe le dit aéroport

Country : Pays dans lequel se situe l'aéroport

Latitude et Longitude : les coordonnées géographiques de l'aéroport

Informations sur les tables (2)

La table Incidents est construite de la manière suivante:

id_incident : clé primaire caractérisant un unique accident

Date : Année de survenue de l'accident

Location : Lieu aproximatif de la survenue de l'accident

Pays : Pays dans lequel est survenu l'accident

Modèle d'avion : modèle d'avion impliqué dans l'accident

Aéroport de départ et Aéroport d'arrivée : clés étrangères se référençant aux tables Airports_dep et Airports_arr

Informations sur les tables (3)

La table Incidents_description est construite de la manière suivante:

id_incident_desciption : clé primaire caractérisant une unique description d'accident

id_incident : clé étrangère référençant l'accident de la description

Type d'accident : code représentant la catégorie de l'accident

Décès : Nombre de personnes décédées lors de l'accident (un nombre inférieur à 0 signifie une absence d'information)

Compagnie : Compagnie effectuant le vol
